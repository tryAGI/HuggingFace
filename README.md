### Usage
```csharp
using HuggingFace;

using var client = new HuggingFaceClient(apiKey, client);
var response = await client.GenerateTextAsync(
    "gpt2",
    new GenerateTextRequest
    {
        Inputs = "Hello",
        Parameters = new GenerateTextRequestParameters
        {
            MaxNewTokens = 250,
        },
    });
```
